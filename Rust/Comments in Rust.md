For [[single line comment]]s we can use:
```rust
// This is a single line comment
```

For [[multiline comment]]s
```rust
/*
    This
    Is a
    Multiline
    Comment
*/
```

```rust
fn main() {
    // a single line comment
    /*
    Multiline
    Comment
    */
}
```
