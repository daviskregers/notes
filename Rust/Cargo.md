Cargo is a [[Rust language]]'s [[build system]] and [[package manager]]. This tool is used to manage rust projects.

To create a new project using cargo:

```
cargo new projectname --bin or --lib
```

To [[compile]] the project:

```
cargo build
cargo check
cargo run
```
