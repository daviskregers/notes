## SSE-S3

-  encrypts [[AWS S3 Objects]] using keys handled & managed by AWS
- [[AWS S3 Objects]] are encrypted server side
- [[AES-256]] encryption type
- Must set header `"x-amz-server-side-encryption": "AES256"`
